Entendiendo el mundo del Desarrollo

Desarrollo Web es un termino que define la creación de sitios web para internet. Para llegar a esto se utilizan tecnologias de software del lado del servidor(desarrollador backend) y del cliente (desarrollador frontend) que involucran una combinacón de procesos de base de datos con el uso de un navegador web para de eso modo realizar determinadas tareas o mostrar información.
Usando lenguajes compilados, semicompalidados o interpretados se crean la funcionalidad como la interfaz de los usuarios, un Desarrollo orientado a la web termina siendo homogenio y multiplataforma, dependiendo de las tecnologías utilizadas puede ser más rapido y robusto tanto para diseñar, implementar y probarlo una vez terminado.
Principalmente el desarrollador web es quien realiza la labor de ocuparse del funcionamiento del software, es tarea del diseñador web preocuparse por el aspecto final de la pagina. Al momento de preguntarnos sobre los lengujaes mas comunes para realizar el desarrollo de las paginas web son HTML, CSS, JAVASCRIP, entre otros, pero aun nos continua faltando una parte importante como es el sistema de gestión de base de datos donde las podemos clasificar desde MySQL, que es la más popular seguida por Oracle y SQL Server dejando en ese orden las mas importantes, esto definido por su nivel de uso en el mercado.
